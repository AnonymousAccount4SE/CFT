Dictionaries
------------

Dictionaries are maps from names (Strings) to values of
any type. They are typically used for representing 
configuration values.


Dict
----

The global Dict() function generates an empty Dict object.
It has function .set("key",value) and .get("key").

Examples:

Dict.set("a",1).get("a")

- Returns 1
